# Sborkai

AI-продуктовый сайт на базе Vite, React, TypeScript, Tailwind CSS и shadcn/ui.

## Локальный запуск

```sh
npm install
npm run dev
```

## Проверки

```sh
npm run check:site-scope
npm run lint
npm run build
```

## Архитектурные заметки

- Бренд, временные контакты и основные ссылки лежат в `src/config/site.ts`.
- Блог, кейсы и отзывы удалены из v1: нет маршрутов, меню, главных секций и исходных файлов этих зон.
- Заявки в первой версии должны вести в живые каналы связи; backend-интеграции не считаются частью текущего запуска.
